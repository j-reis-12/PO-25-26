# Object-Oriented Programming course @ IST

**Project - Library Management System**

Goal: Develop a fully object‑oriented library management system capable of handling creators, works, users, loans,
notifications, time progression, and persistent state. The project emphasizes strong OO design principles, especially
extensibility and the open‑closed principle, requiring the system to support new work types, new user
classifications, new borrowing rules, and new search mechanisms with minimal impact on existing code. It integrates
domain modeling, inheritance, polymorphism, exception handling, serialization, and a structured UI command framework
to deliver a robust, modular, and maintainable application.
